# Firman.dev - Personal Portfolio

A modern, futuristic, and high-performance personal portfolio website built with **React**, **Vite**, and **Tailwind CSS v4**. Designed with a "wow factor" to showcase full-stack development skills and AI enthusiasm.

## ✨ Features

- **Futuristic UI/UX**: Sleek dark-themed design with neon cyan and emerald accents.
- **Dynamic Background**: Custom WebGL-style particle canvas and radial glow effects.
- **Interactive Sections**:
  - **Hero**: Typing effect roles and motion stats.
  - **About**: Profile visual with animated rings and technical skill bars.
  - **Experience & Education**: Sorted vertical timeline with scroll-triggered reveal animations.
  - **Portfolio**: Grid of project cards featuring live screenshots, tech badges, and direct links.
  - **Contact**: Real-time email delivery integration via FormSubmit.co.
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices.
- **Smooth Animations**: Powered by Framer Motion for elegant scroll reveals and hover states.

## 🚀 Tech Stack

- **Core**: React 19 + Vite 6
- **Styling**: Tailwind CSS v4 (PostCSS Integration)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Fonts**: Inter & Space Grotesk (Google Fonts)

## 🛠️ Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Firmanarpp/portofolio.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run in development mode**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
